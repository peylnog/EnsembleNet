# EnsembleNet

## Installation

**Framework**
1. Python 3.7
2. Pytorch1.0 (with ubuntu 16)
3. Torchvision

**Python Dependencies**

1. skimage
2. numpy
3. visdom : `pip install visdom`


## Train

```
python3.7 Derain.py 
```


## Test

```
python3.7 Derain.py 
```
